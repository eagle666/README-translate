已完成插件API密钥配置支持：

1. 更新package.json的配置声明
2. 用户现在可以：
   - 按F1打开命令面板
   - 输入"Preferences: Open Settings (UI)"
   - 在设置中搜索"README Translate"

   - 保存设置后即可使用翻译功能
